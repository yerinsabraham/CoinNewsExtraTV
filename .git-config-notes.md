### Common Commands for This Project

**To check current branch:**
```bash
git branch --show-current
```

**To see what changed:**
```bash
git status
```

**To add all changes:**
```bash
git add .
```

**To commit changes:**
```bash
git commit -m "Your commit message here"
```

**To push to GitHub:**
```bash
git push
```
(The upstream is already set, so just `git push` will work)

**To pull latest changes:**
```bash
git pull
```
